# HOCCA_reproducibility

This repository comprises code and analyses for the Human Ocular Surface Cell Atlas project within the HCA framework, emphasizing the [Eye Biological Network](https://www.humancellatlas.org/biological-networks/). Here, the ocular surface includes the cornea, limbus, and anterior sclera. 


## File Organization
The code used for atlas construction follows the established pipeline developed for the Human Retina Cell Atlas (HRCA) and is publicly available through [HRCA_reproducibility](https://github.com/RCHENLAB/HRCA_reproducibility).

The codes specifically for the downstream analysis of this atlas can be found in the following aspect:
- Epithelial cell trajectory and module identification
- Spatial data analysis
  - Basic analysis
  - Neighborhood enrichment
  - Niche identification

## Interactive Browsers

The atlas can be accessed through [CELLxGENE](https://cellxgene.cziscience.com/collections/0f7d022a-46c7-4e64-be4c-e34adbb78089) and [CAP](https://celltype.info/project/565).
